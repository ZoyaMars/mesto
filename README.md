# Проект: Место

Интерактивная страница, на которой можно: добавлять картинки, удалять картинки, ставить лайки, а также менять имя профиля.

**Использованы технологии:** 
* HTML / CSS
* Figma
* БЭМ
* Адаптивная верстка
* Валидация форм при помощи JS.
* Использованы ES6-классы
* ООП
* Webpack
* JavaScript
